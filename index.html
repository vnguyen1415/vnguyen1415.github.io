Emergency Call 911 (2015-2017)¶

By Vinh T. Nguyen¶

This tutorial will walk you through the entire data science pipeline starting from data collection and processing, then moving on to exploratory data analysis and data visualization.

This tutorial will based on Emergency(911) calls including fire, traffic, and EMS for montgomery County, PA in the time period end of 2015-2017, and use hypothesis and proving my analysis on the dataset. Lastly, we will show the messages covering insight learnied during the tutorial. However, this tutorial will focusing more on the data processing and analysis using visualization.

3/ Machine learning¶

We will try using these data and train the machine to find out about the pattern behind the data. Based on the graph that we did, we can predict the amount of phone calls in certain hours .

We want to predict number of phone calls by hours. So we will use numpy polyfit totrain a polynomial fit for those 2 y and x values.

Numpy polynomial will train on a set of x and y points and a degree n. It fit a polynomial p(x) = p[0] x deg +.....+ p[deg] of degree deg to points(x,y). It returns a vector of coefficients p that minimises the squared error. For more information, you can find at https://docs.scipy.org/doc/numpy-1.13.0/reference/generated/numpy.polyfit.html

We'll shuffle our data and then split them into testing and training. We'll choose 1/3 or our data ast testdata.
